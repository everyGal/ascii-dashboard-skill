## Spec Template
```json
{
  "id": "(required) string вҖ” unique snake_case identifier",
  "type": "treemap",
  "title": "(required) string вҖ” display name",
  "category_field": "(required) string вҖ” dimension field for rectangles",
  "value_field": "(required) string вҖ” measure controlling rectangle size",
  "parent_field": "(optional) string вҖ” parent dimension for nested treemap",
  "label_field": "(optional) string вҖ” field for rectangle labels",
  "grid": {
    "col": "(required) int 1-12",
    "row": "(required) int",
    "width": "(required) int вҖ” typically 5-8",
    "height": "(required) int вҖ” typically 2"
  }
}
```

## Sizing Guidelines
- **Width**: 5-8 columns (needs space to show proportional rectangles clearly)
- **Height**: 2 rows
- **Common pattern**: Paired with a line chart (6+6), or standalone at 8+ columns
- **Best for**: Part-to-whole hierarchy, budget allocation, market share breakdown
