## Spec Template
```json
{
  "id": "(required) string вҖ” unique snake_case identifier",
  "type": "network",
  "title": "(required) string вҖ” display name",
  "node_field": "(required) string вҖ” dimension for nodes",
  "edge_source_field": "(required) string вҖ” source node for each edge",
  "edge_target_field": "(required) string вҖ” target node for each edge",
  "node_size_field": "(optional) string вҖ” measure controlling node size",
  "edge_weight_field": "(optional) string вҖ” measure controlling edge thickness",
  "grid": {
    "col": "(required) int 1-12",
    "row": "(required) int",
    "width": "(required) int вҖ” typically 6-8",
    "height": "(required) int вҖ” typically 2-3"
  }
}
```

## Sizing Guidelines
- **Width**: 6-8 columns (roughly square aspect ratio)
- **Height**: 2-3 rows
- **Best for**: Dependency graphs, org charts, collaboration networks, flow analysis
- **Complexity note**: High complexity вҖ” requires pre-computed node positions; force-directed layout cannot be computed in most charting tools without a dedicated library
