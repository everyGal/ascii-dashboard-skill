## Spec Template
```json
{
  "id": "(required) string",
  "type": "waterfall",
  "title": "(required) string",
  "category_field": "(required) string вҖ” dimension for each bar segment",
  "value_field": "(required) string вҖ” metric (positive = increase, negative = decrease)",
  "total_label": "(optional) string вҖ” label for the final total bar, default: 'Total'",
  "show_connectors": "(optional) bool вҖ” lines connecting bars, default: true",
  "grid": {
    "col": "(required) int",
    "row": "(required) int",
    "width": "(required) int вҖ” typically 6-8",
    "height": "(required) int вҖ” typically 2"
  }
}
```

## Sizing Guidelines
- **Width**: 6-8 columns
- **Height**: 2 rows
- **Best for**: Revenue bridges, profit breakdowns, budget variance analysis
