## Spec Template
```json
{
  "id": "(required) string вҖ” unique snake_case identifier",
  "type": "packed_bubble",
  "title": "(required) string вҖ” display name",
  "category_field": "(required) string вҖ” dimension for bubble labels",
  "value_field": "(required) string вҖ” measure controlling bubble size",
  "color_field": "(optional) string вҖ” dimension or measure for bubble color",
  "grid": {
    "col": "(required) int 1-12",
    "row": "(required) int",
    "width": "(required) int вҖ” typically 4-6",
    "height": "(required) int вҖ” typically 2"
  }
}
```

## Sizing Guidelines
- **Width**: 4-6 columns (bubbles need roughly square space)
- **Height**: 2 rows
- **Common pattern**: Beside a table or bar chart (6+6) for complementary detail, or standalone at 6 columns
