## Spec Template
```json
{
  "id": "(required) string вЂ” unique snake_case identifier",
  "type": "calendar_heatmap",
  "title": "(required) string вЂ” display name",
  "date_field": "(required) string вЂ” date dimension for calendar cells",
  "value_field": "(required) string вЂ” measure controlling color intensity",
  "color_scale": "(optional) string вЂ” sequential | diverging, default: sequential",
  "time_range": "(optional) string вЂ” month | quarter | year, default: month",
  "grid": {
    "col": "(required) int 1-12",
    "row": "(required) int",
    "width": "(required) int вЂ” typically 6-8",
    "height": "(required) int вЂ” typically 2-3"
  }
}
```

## Sizing Guidelines
- **Width**: 6-8 columns (needs room for 4-5 week columns plus day labels)
- **Height**: 2-3 rows (7 day-of-week rows plus title and legend)
- **Common pattern**: Full-width (12) for year view, 6-8 for monthly view beside a summary panel
