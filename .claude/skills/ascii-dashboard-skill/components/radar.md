## Spec Template
```json
{
  "id": "(required) string вЂ” unique snake_case identifier",
  "type": "radar",
  "title": "(required) string вЂ” display name",
  "axes": "(required) array of strings вЂ” field names for each axis (minimum 3)",
  "category_field": "(required) string вЂ” dimension to compare (e.g., product, team)",
  "normalize": "(optional) bool вЂ” normalize all axes to 0-1 scale, default: true",
  "show_grid": "(optional) bool вЂ” show concentric grid rings, default: true",
  "grid": {
    "col": "(required) int 1-12",
    "row": "(required) int",
    "width": "(required) int вЂ” typically 4-6",
    "height": "(required) int вЂ” typically 2-3"
  }
}
```

## Sizing Guidelines
- **Width**: 4-6 columns (square aspect ratio works best)
- **Height**: 2-3 rows
- **Best for**: Multi-dimensional comparisons, performance profiles, competitive analysis
- **Complexity note**: High implementation complexity вЂ” requires trigonometric coordinate transformations
